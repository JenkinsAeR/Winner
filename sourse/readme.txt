winners.py - главный файл со всей логикой
winners_ui.py - графический интерфейс главного окна
modal.py - графический интерфейс модального окна
images_rc.py - файл с картинкой закрытия программы в текстовом формате
close.png - картинка крестик(необязательна)
medal.ico - иконка программы "медалька" (необязательна)
Winners.ui - Файл для QtDesigner, главное окно. (необязательно)
Modal.ui - Файл для QtDesigner, модальное окно окно. (необязательно)
participants.txt - стандартный файл, который используется по умолчанию.
Из него берутся данные для демонстрации, при нажатии 'выбрать победителей'(необязательно)
